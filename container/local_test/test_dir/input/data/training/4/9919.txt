TurboTax Deluxe 2014 Fed + State + Fed File was a very easy to use software that helped you though each step.  It was you some questions and you just enter the information.  If you had not used any TurboTax before then it creates a folder for you and places your information in it.  If you have use it before, then it will import your information for you.  What I did not like about it is that it will not efile in some cases.
